이 플러그인은 "다음뷰"에 포스트를 자동 송고 및 송고된 포스트 하단에 추천 버튼을 출력 합니다.

== Description ==

blueDaumView 플러그인은 포스트 작성시 메타박스에 있는 다음뷰 송고 리스트 박스에서 송고할 카테고리를 선택한후 포스트 발행과 동시에 다음뷰에 포스트를 송고하여 줍니다.

또한, 다음뷰로 송고된 포스트에 한해 포스트 하단에 추천 버튼을 출력합니다. 추천 버튼의 형태(박스형, 얇은 박스형, 버튼형)는 관리자 툴에서 설정 할 수 있으며 출력하는 스킨을 동시에 설정 할 수 있습니다.

== Installation ==

플러그인을 설치하기 위해서는 아래와 같이 순서대로 진행하시면 됩니다.

1. blueDaumView를 다운로드 받습니다.
2. 다운로드 받은 플러그인을 워드프레스의  플러그인 디렉토리에서 압축을 풉니다.
3. 관리자 페이지의 플러그인 설정에서 blueDaumView를 활성화 시킵니다.
4. 설정 메뉴의 blueDaumView 메뉴에서 자신에게 맞는 추천 박스 형태와 스킨을 설정합니다.

자동 설치에 관련된 설명서는 http://www.blueiblog.com/ko/archives/2261 에
업데이트 되었습니다. 설치가 어려우신분들은 한번씩 읽어보신 후 설치 해 주세요.
